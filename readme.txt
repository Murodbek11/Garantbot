TRUSTIX — Telegram Anti-Scam Бот

Команды (для владельца и модераторов):
/add @user причина — добавить скамера
/suspect @user причина — подозрительный
/fakegarant @user причина — фейк гарант
/garant @user @канал_отзывов — проверенный гарант
/remove @user — удалить из базы
/check @user — проверить статус
/garantlist — список проверенных гарантов

Файл базы: data/antiscam.json
